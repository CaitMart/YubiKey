# YubiKey Setup 
Yubikey is compatible to many different websites [compatibility list](https://www.yubico.com/works-with-yubikey/catalog/?sort=popular). Below is how I was able to create 
passkey's on several different accounts to use my Yubikey for MFA. 

## Google 
For Google I was able to go to my account settings > security and sign-in > passkeys and security keys.
From there you then plug in the yubikey and add it to the account. You press on the yubikey to verify and it is then attatched to your account.

## Microsoft 
When logging on to Microsoft it gave me the option to create a passkey. With my Yubikey plugged in, I was able to connect it to my microsoft account to use it
during login. You can manage verification options in your security settings in your Microsoft account to add or remove.

## MacOS  
You can add your YubiKey on your MacOS by going into system settings > click account name > sign-in and security > two-factor authentication > security keys > set up 
Note you must have two keys present to use it on macOS because a backup key is needed incase you lose your main one.
